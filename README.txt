===== Interest Calendar Ledger Application =====

This is a portable application that manages interest calculations and client information.

== RUNNING THE APPLICATION ==

1. Simply double-click on "run_app.bat"
2. The script will automatically:
   - Install Python if needed (no administrator rights required)
   - Set up the necessary environment
   - Install required dependencies
   - Launch the application

3. Once launched, the application will open in your default web browser
   - If it doesn't open automatically, navigate to: http://localhost:8501

== REQUIREMENTS ==

- Windows 7 or newer
- Internet connection (first run only, for downloading dependencies)
- 500MB of free disk space
- No administrator privileges required

== TROUBLESHOOTING ==

If you encounter issues:

1. Make sure you have an active internet connection for the first run
2. If a Windows Defender SmartScreen warning appears, click "More info" and "Run anyway"
3. The command window must remain open while the application is running
4. To close the application completely, close the command window

== DATA STORAGE ==

All your data is stored locally in the "data" folder within this directory.
No data is sent to external servers.
